# Setting Up Git

## Install Git
- Command used to verify Git is installed:
  - `git --version`
- Result:
  - `git version 2.43.0`

## Configure username and email
- Commands:
  - `git config --global user.name "YOUR_NAME"`
  - `git config --global user.email "YOUR_EMAIL"`
- Verification:
  - `git config --global --list`
